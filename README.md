# wechat-logo-simulator (微信小程序 Logo 视觉模拟器)

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

这是一个专为独立开发者和 UI 设计师打造的**纯本地、极简微信小程序 Logo 视觉验证与裁剪工具**。

### 🚀 解决什么痛点？
微信官方对小程序头像限制极严——**每年仅有 12 次宝贵的修改机会**！

在项目实际交付或上线前，产品经理、设计师和开发者经常面临以下尴尬：
* **盲盒式上传**：无法提前得知图标在真机（发现页列表、聊天列表、下拉任务栏）上的**真实视觉大小**，字号小了看不清，字号大了显得臃肿。
* **试错代价高昂**：为了测试图标文字是否被官方 1:1 圆形遮罩切边，或者为了微调尺寸，不得不白白烧掉珍贵的年度修改次数。
* **暗色边缘“抓瞎”**：无法预知图标在微信下拉任务栏（深色背景）下是否会产生边界模糊、对比度不足的问题。

本工具让你**在本地即可零成本、无限次地进行 1:1 真机多场景效果验证**。把修改机会留给真正的重大品牌升级，而不是无谓的视觉试错！


---

### ✨ 核心功能
* **📐 原生 1:1 裁剪**：内置微信原生裁剪逻辑，锁定 1:1 比例，确保选图中心点不偏移。
* **📱 真实场景模拟**：
  * 模拟小程序在**微信下拉任务栏**中的展示效果。
  * 支持**暗色背景/暗色边缘**视觉验证，确保在深色模式下依然清晰。
* **⚡ 纯本地运行**：不依赖任何后端服务器，图片处理均在小程序本地 Canvas 完成，安全高效。

---

### 🛠️ 开发与运行环境
* **主要语言**：JavaScript (100%)
* **基础库版本**：推荐使用 `3.16.1` 或以上
* **适配机型**：已针对 iPhone 12/13 等主流刘海屏及全面屏进行底部安全区与下拉栏深度适配。
